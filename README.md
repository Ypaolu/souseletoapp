Projeto desenvolvido para a ONG SouSeleto (Guaramirim/SC) com o objetivo de digitalizar a gestão das escolinhas de futebol. O aplicativo substitui o controle físico (papel e caneta) por uma solução mobile ágil e integrada.

Professores e funcionários da ONG perdiam tempo precioso com anotações manuais e burocracia física, o que dificultava a análise de presença e a organização de eventos esportivos.

Principais funcionalidades:
- Chamada: Professores podem fazer as chamadas diretamente pelo smartphone.
- Convocações: Organização simplificada de jogadores para campeonatos.
- Ponto Eletrônico: Controle de jornada para os funcionários da ONG.
- Perfil do Aluno: Acesso rápido a informações básicas e histórico de presença.

Tecnologias Utilizadas:
- Flutter: Framework para desenvolvimento mobile nativo (Android/iOS).
- Firebase Auth: Autenticação segura via E-mail e Senha.
- Cloud Firestore: Banco de dados NoSQL para sincronização de dados em tempo real.
- Dart: Linguagem de programação robusta para a lógica de negócio.

Segurança e Configuração
Este repositório não contém arquivos de credenciais (google-services.json, firebase_options.dart) por motivos de segurança e boas práticas de desenvolvimento.
Para rodar o projeto:
- Configure um projeto no Firebase Console.
- Ative o método de autenticação Email/Password.
- Utilize o FlutterFire CLI para gerar seu arquivo de configuração na pasta lib/.
